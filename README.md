# 祖安语录

一个单页吐槽生成器，支持随机输出、模式切换、复制、后退，并适配移动端。

## 功能
- 火力全开 / 口吐莲花
- 骂爹模式（仅筛选包含“爹”的语录）
- 一键复制
- 双击输出区域可再生成

## 目录结构
```
.
├── index.html
├── styles.css
├── script.js
├── 祖安语录.txt
├── 祖安语录2.txt
└── .nojekyll
```

## 本地预览
注意：直接用 `file://` 打开会导致 `fetch` 读取 txt 失败。请使用本地服务器：

```bash
python -m http.server 8080
```

访问 `http://localhost:8080`。

## 词库
- 将语录按行写入 `祖安语录.txt` 和 `祖安语录2.txt`
- 建议使用 UTF-8 编码

## GitHub Pages
仓库已可直接部署 Pages（`main` 分支 / 根目录）。如需自定义域名，在仓库根目录添加 `CNAME` 文件并配置 DNS。
